\hypertarget{index_autotoc_md0}{}\doxysection{\texorpdfstring{SFML Game Engine}{SFML Game Engine}}\label{index_autotoc_md0}
This is a simple game engine built using the SFML (Simple and Fast Multimedia Library) in C++. It provides a basic framework for creating 2D games with support for keyboard and mouse input, text rendering, and basic event handling. The engine is designed to be easy to use and extend, making it a great starting point for small-\/scale game development projects.\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Window creation and management
\item Basic event handling for keyboard and mouse input
\item Text rendering with customizable fonts and colors
\item Button functionality for specific actions
\item Support for setting the background color of the window
\item FPS (Frames Per Second) control for smooth animation
\item Logging of errors to both the console and a log file
\item Polyline drawing with support for open and closed shapes
\item Elapsed time display
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Dependencies}{Dependencies}}\label{index_autotoc_md2}
This project relies on the following\+:


\begin{DoxyItemize}
